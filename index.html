Overview

Ask! is a WASM contract language framework based on the AssemblyScript that runs on the blockchains built on the Substrate framework


With this documentation portal we want to provide comprehensive documentation for anything you need to work with Ask!.

If you spot gaps in the information provided, or are uncertain about any specific area, please do log an issue or make a pull-request.

We want to have good documentation so that people can be productive right away.
